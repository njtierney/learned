The Data Cleaning Document

Onboarding

Hi there! Welcome to the group! This is “The Data Cleaning Document”. Here’s the process

  1. Read the data in
  2. Go through various ad hoc tests
  3. Make sure the data is “clean”
  4. Repeat for subsequent years of data
  5. Produce a plot of the num,ber of people studying in each age group in each state
  6. Produce a box plot, and a table that shows The 5 number summary (min, 1st quartile, median, 3rd quartile, max) of number educated for each state.
  7. Produce a table stating the summary of proportion educated in each state

When we get another dataset, we make a new document to clean that data. The process is:

  1. Copy this document
  2. Update the data
  3. Check it is “clean”
  4. Make the same plot

It might seem repetitive, and there’s often little sneaky bits in there, but it’s just part of the process! Enjoy!

fixed_age_group_education_2014 <- education_2014 |>
  mutate(
    age_group = str_replace_all(
      age_group,
      pattern = "---|--|-",
      replacement = "_"
    ),
    age_group = as.factor(age_group)
  )
