# learned

<!-- badges: start -->

<!-- badges: end -->

The goal of learned is to demonstrate some cleaning tools for education
data.

## Installation

You can install the development version of learned from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("njtierney/learned")
```
